# RecipeHub 🍽️

RecipeHub is a responsive frontend recipe book website built using HTML, CSS, and JavaScript.

## Features

* Browse featured recipes
* Search recipes by title
* Search recipes by ingredients
* Filter recipes by category
* View detailed recipe pages
* Responsive design for mobile, tablet, and desktop devices

## Categories

* Breakfast
* Lunch
* Dinner
* Dessert

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Browse recipes, search, filter, and view detailed recipe information.

## Project Objective

This project was developed as part of a Frontend Development Internship Task. The objective was to create a frontend-only recipe book website where users can browse recipes, search by ingredients or title, filter recipes by category, and view complete recipe details.
